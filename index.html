THE PROBLEM

창작자가 매일 겪는 3가지 고통

레퍼런스 탐색은 창작의 시작점이지만, 현실은 막막합니다.

01

저작권 위험

Google 이미지, Pinterest 등에서 무분별하게 참조하다 법적·윤리적 문제에 노출됩니다. 창작물이 표절 시비에 휘말리는 순간 커리어 전체가 위험해집니다.

"어디까지가 참고이고 어디서부터 표절인지 경계가 모호해요."
02

낮은 매칭 정확도

키워드 검색으론 원하는 구도·포즈를 찾기 불가능합니다. "팔 들고 오른쪽 보는 인물" 같은 구체적 조건을 텍스트로 설명할 수 없어 수십 분을 낭비합니다.

"머릿속에 그림은 있는데, 그걸 설명할 말이 없어요."
03

시간 낭비

레퍼런스 수집에만 하루 평균 1-2시간이 소요됩니다. 창작에 써야 할 황금 같은 시간이 검색과 저장에 사라집니다.

"그림 그리는 시간보다 레퍼런스 찾는 시간이 더 길어요."
THE SOLUTION

Sketch-Link는
구조로 검색합니다

AI를 복제 도구가 아닌 '검색과 분석'의 도구로 사용합니다. 화풍(Style)이 아닌 구조(Structure)와 형태(Form) 중심의 유사도 계산으로, 창작자에게 진정으로 유용한 레퍼런스를 제공합니다.

  • 100% 클린 데이터 CC0 및 퍼블릭 도메인만 인덱싱 — 법적 위험 제로
  • 구조 기반 AI 매칭 포즈·구도·레이아웃을 시각적으로 분석하는 독자적 알고리즘
  • 3초 이내 결과 스케치 업로드 후 즉시 최적 레퍼런스 매칭 완료
  • 50만+ 이미지 DB Unsplash, Met Museum, Europeana 등 세계 유수 공공 아카이브

HOW IT WORKS

단 3단계로 완성

복잡한 설정 없이, 스케치만 있으면 됩니다.

01

스케치 업로드

러프 드로잉, 연필 스케치, 심지어 냅킨 위 낙서도 괜찮습니다. 파일 업로드 또는 캔버스에 직접 그려보세요.

JPG · PNG · SVG 드래그&드롭 직접 그리기
02

AI 구조 분석

엣지 감지·포즈 추정·레이아웃 분석 알고리즘이 스케치의 구조와 형태를 수치화합니다. 화풍은 무시, 뼈대만 봅니다.

엣지 감지 포즈 추정 레이아웃 분석
03

레퍼런스 매칭

50만+ CC0·퍼블릭 도메인 이미지 DB에서 유사도 순으로 결과를 제공합니다. 모든 결과물은 법적으로 안전하게 사용 가능합니다.

유사도 순 정렬 CC0 인증 원클릭 저장
FEATURES

창작자를 위해 설계된 모든 것

MVP부터 완성형까지, 단계별로 쌓아가는 기능들입니다.

MVP 핵심

구조 기반 시각 검색

키워드가 아닌 스케치의 구조적 특징(엣지, 포즈, 레이아웃)으로 이미지를 검색합니다. 텍스트로 표현할 수 없는 비주얼을 정확하게 찾아냅니다.

MVP 핵심

CC0 클린 데이터베이스

Unsplash, Pexels, Met Museum, Europeana 등 신뢰할 수 있는 소스에서만 수집한 퍼블릭 도메인 이미지만 제공합니다.

Unsplash Met Museum Europeana NYPL
MVP 핵심

스마트 필터링

카테고리(인물·풍경·오브젝트), 시대, 스타일 등 다양한 조건으로 결과를 좁혀 원하는 레퍼런스를 빠르게 찾습니다.

2단계

레퍼런스 보드

저장한 레퍼런스를 프로젝트별로 정리하고, 팀과 공유할 수 있는 무드보드 기능입니다.

2단계

크리에이터 커뮤니티

창작자들이 레퍼런스 보드와 프로세스를 공유하는 커뮤니티. 서로의 작업 방식에서 영감을 얻습니다.

3단계

플러그인 & API

Photoshop, Procreate, Figma 등 크리에이터가 이미 사용하는 도구 안에서 Sketch-Link를 바로 사용할 수 있는 플러그인 생태계를 구축합니다.
